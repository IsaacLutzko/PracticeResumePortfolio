Link To Web App Here!

This web app calculates statistics on the text that is inserted into the text box at the top - from there a user can select the 'Compute Stats' button and the statistics will be shown. The information of the text that is in the text box shows the number of characters, the number of words, the number of lines, the number of non-empty lines, the average word length, the maximum line length, the 10 longest words, and the most frequent words in the text.
